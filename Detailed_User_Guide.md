Witness The Fitness (WTF) is a comprehensive fitness management system designed to streamline operations for fitness centers and personal trainers. This guide provides detailed instructions for all user roles within the system.

### Key Features
- **Role-based access control** (Admin, Personal Trainer)
- **Client management** with detailed profiles
- **Session scheduling and tracking**
- **Payment processing** and financial reporting
- **Performance analytics**
- **Automated notifications**

### Trainer Management
#### Adding a New Trainer
1. Navigate to **Trainers** > **Add New**
2. Complete all required fields:
   - Full Name
   - Email Address
   - Phone Number
   - Specialization
   - Profile Picture (optional)
3. Set initial login credentials
4. Assign appropriate permissions
5. Click **Create Account**

#### Trainer Performance Monitoring
1. Go to **Dashboard** > **Trainer Analytics**
2. View key metrics:
   - Client retention rate
   - Session completion rate
   - Revenue generated
   - Client satisfaction scores
3. Export reports as needed

### Client Management (Admin View)
1. **Adding a New Client**
   - Navigate to **Clients** > **Add New**

   - Assign to a trainer
   - Set up initial payment plan
   - Upload any relevant documents

2. **Client Overview**

## â“ Frequently Asked Questions

### For Administrators
**Q: How do I reset a trainer's password?**
A: Navigate to Trainers > Select trainer > Account > Reset Password

**Q: Can I export client data for backup?**
A: Yes, go to Settings > Data Management > Export

### For Trainers
**Q: How do I update my availability?**
A: Go to Profile > My Schedule > Update Availability

**Q: What should I do if a client is late?**
A: You can adjust the session time or mark as a late arrival in the session notes
