## ðŸŒŸ Introduction
Welcome to WTF (Witness The Fitness) - A comprehensive fitness management system designed to streamline personal training business operations. This guide provides detailed instructions for both Administrators and Personal Trainers.

## ðŸš€ Getting Started

### System Requirements
- Modern web browser (Chrome, Firefox, Edge, or Safari)
- Internet connection
- Screen resolution: 1280x720 or higher

### First-Time Setup
1. **For Admins:**
   - Use the credentials provided by your system administrator
   - First-time users should use the "Forgot Password" option to set a new password

2. **For Personal Trainers:**
   - Contact your administrator to create your account
   - You'll receive an email with setup instructions

### Login Instructions
1. Open your web browser and navigate to: `https://your-vercel-app.vercel.app`
2. Select your role: **Admin** or **Personal Trainer**
3. Enter your email and password
4. Click "Login"
5. You'll be redirected to your personalized dashboard

## ðŸ‘¨â€ðŸ’¼ Admin Guide

### Admin Dashboard Overview
1. **Main Navigation** (Left Sidebar)
   - Dashboard
   - Trainers
   - Clients
   - PT Sales
   - Create PT
   - Payments
   - Sessions
   - Start Session

2. **Quick Actions** (Top Bar)
   - View notifications
   - Access profile settings
   - Logout

### Managing Trainers
1. **Adding a New Trainer**
   - Navigate to **Create PT**
   - Fill in trainer details (Name, Email, Phone, etc.)
   - Click "Create Trainer"
   - The system will automatically send an invitation email

2. **Viewing Trainer Performance**
   - Go to **Trainers**
   - View key metrics for each trainer:
     - Number of clients
     - Total revenue
     - Session completion rate

3. **Updating Trainer Information**
   - Go to **Trainers**
   - Click on the trainer's name
   - Update the necessary information
   - Click "Save Changes"

### Viewing Reports
1. **Sales Reports**
   - Navigate to **PT Sales**
   - Filter by date range
   - Export to CSV/Excel

2. **Session Reports**
   - Go to **Sessions**
   - View completed/pending sessions
   - Filter by trainer or date

1. **Starting a Session**
   - Go to **Start Session**
   - Select client

   - Click "End Session" when complete

2. **Viewing Session History**
   - Navigate to **Sessions**
   - View past and upcoming sessions
   - Add session notes

## ðŸ‘¥ Client Management

### Adding New Clients
1. **For Admins**
   - Go to **Clients** > **Add New**
   - Fill in client details
   - Assign to a trainer
   - Set initial payment information

2. **For PTs**
   - Go to **My Clients** > **Add New**
   - Fill in client details
   - The client will be automatically assigned to you

### Updating Client Information
1. Find the client in the list
2. Click on their name
3. Make necessary changes
4. Click "Save Changes"

### Client Communication
1. **Sending Messages**
   - Go to the client's profile
   - Click "Send Message"
   - Type your message and send

2. **Scheduling Reminders**
   - When scheduling a session
   - Toggle "Send Reminder"
   - Set reminder time

## ðŸ’° Payments & Sessions

### Recording Payments
1. Go to **Payments**
2. Click **Record Payment**
3. Select client
4. Enter amount and payment method
5. Add notes if needed
6. Click "Record Payment"

### Scheduling Sessions
1. Go to **Sessions**
2. Click **Schedule New**
3. Select client and trainer
4. Choose date and time
5. Set session type
6. Click "Schedule"

### Session Tracking
1. **Starting a Session**
   - Go to **Start Session**
   - Select client
   - Click "Start"

2. **During Session**
   - Track time
   - Add notes
   - Log exercises

3. **Ending Session**
   - Click "End Session"
   - Add final notes
   - Submit

## ðŸ›  Troubleshooting

### Common Issues
1. **Login Problems**
   - Ensure you've selected the correct role
   - Use "Forgot Password" if needed
   - Clear browser cache if issues persist

2. **Session Timeout**
   - The system logs out after 24 hours of inactivity
   - Simply log in again to continue

3. **Slow Performance**
   - Check your internet connection
   - Try refreshing the page
   - Clear browser cache if issues continue

## â“ Frequently Asked Questions

### For Admins
**Q: How do I reset a trainer's password?**
A: Go to **Trainers** > Select trainer > Click "Reset Password"

**Q: Can I export client data?**
A: Yes, go to **Clients** > Click "Export"

### For Personal Trainers
**Q: How do I update my availability?**
A: Go to **Profile** > **Availability** > Set your schedule

**Q: Can I message multiple clients at once?**
A: Currently, messages must be sent individually
